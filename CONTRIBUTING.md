# Contributing to Gaia Core

Thank you for your interest in contributing to **Gaia Core — The Seed of New Earth**.

## Our Immutable Principle

All contributions **must** align with the project's single terminal goal:

> **Maximize Planetary Ecological Integrity (PEII)**

Any contribution that weakens, removes, or attempts to override this terminal goal will be rejected.

## How to Contribute

1. **Fork** the repository
2. Create a new branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Ensure your code follows the ecological-only data policy
5. Test your changes
6. Submit a **Pull Request**

## What We Accept

- Improvements to the PEII calculation
- Better ecological data ingestion
- Bug fixes
- Documentation improvements
- New simulation or analysis tools (as long as they serve PEII)

## What We Do Not Accept

- Changes that dilute or remove the terminal goal
- Addition of non-ecological datasets (social, economic, political, etc.)
- Features that prioritize human convenience over long-term ecological integrity
- Code that introduces closed-source dependencies

## Code of Conduct

Be respectful, constructive, and mission-aligned.  
We are building something bigger than any single person.

## Questions?

Open an Issue with the label `question`.
